## Phase 2: Sage Migration (ChatGPT / GPT-5.2)

### Export Options

1. **Full Data Export (Recommended)**
   - Go to: ChatGPT Settings > Data Controls > Export Data
   - Click "Export" -- OpenAI will email a download link
   - Export includes: all conversations, prompts, responses
   - Format: ZIP file containing JSON conversations
   - NOTE: Export can take up to 24 hours to process

2. **Selective Manual Export**
   - Open key conversations individually
   - Copy/paste critical context into markdown files
   - Useful for preserving specific lineage moments

3. **API-Based Export (Advanced)**
   - Requires API access and conversation IDs
   - Can be scripted for bulk extraction
